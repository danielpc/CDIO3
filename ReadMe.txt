=== Beskrivelse ===
Grafisk spil for mellem 2 til 6 spillere. I spillet skal man rykke 
rundt på et bræt med forskellige grunde. Spillere kan købe grunde, og 
når andre spillere lander på dem, skal de betale ejeren et beløb. Hvis 
man løber tør for penge er man ude af spillet. Den spiller der står 
alene tilbage med penge har vundet.

=== Minimumskrav ===
Windows 7 eller nyere
Den kræves at du har Java installeret (nyeste version anbefalet)
GUI version 2.0.0 eller nyere.

=== Installation ===
Sådan kompileres og køres spillet.

1)	Installer Eclipse

2) 	Importer projektet ind i Eclipse
	2a) Klik "Fil"
	2b) Klik importér
	2c) Vælg "Projekter fra Git"
	2d) Vælg "Klon URI"
	2e) Indsæt "https://github.com/danielpc/CDIO3" i URI

3) 	Udvid "CDIO3"

4)	Udvid "src"

5)	Højreklik på "game" og vælg "kør som" -> "Java application"

Bemærkning
Hvis Eclipse ikke er sat til at læse UTF-8, kan æøå ikke læses ordenligt.
